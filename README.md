# ImageLabel - DL 图像标注标签修改工具
本工具用于修改已经标注好框的深度学习目标检测数据集。

由于在实际项目中，同一个标注框可能会用于多个分类训练中，比如工人可能同时属于“戴安全帽”类和“穿马甲”类。使用此软件可以快速修改目标框类别，而不必重复标注。

## Prepare
一个图像数据集文件夹，以及一个标注框 txt 文件，格式如下：

```
01_00011.jpg UnusingMobile 0 512 36 673
01_00012.jpg UnusingMobile 0 516 37 679
01_00013.jpg UnusingMobile 2 516 48 687
01_00014.jpg UnusingMobile 8 515 53 693
```

## 使用软件
### 0 修改新类别
在 `ClassList.txt` 中修改你要新修改的类别。

### 1 载入
在文件区分别载入`数据集目录`和`标注框 txt 文件`。

点击“初始化标注文件(慎)”会将所有框的类别更改为 `ClassList.txt` 中的类别。

### 2 标注
点击“上一张”或“下一张”翻页，点击图片中任意框，会自动修改该框的类别，并变色。也可以在右侧浏览、修改每一个框的类别。

如果框挡住了视线，可以“暂时隐藏框”。
